Mansão Assombrada

html / css javascript

Jogo

Este projeto foi desenvolvido durante as disciplinas de Práticas de Design e Padrões e Projetos de Sítios para Internet 2, ministradas pelo Professor Leonardo Souza de Lima no curso de Sistemas para Internet da Fatec Araras, com o objetivo de aplicar os conceitos de design e programação em conteúdos web.

A indústria de jogos eletrônicos tem experimentado um notável crescimento e se tornado cada vez mais presente na vida das pessoas. Com a capacidade de publicar jogos gratuitamente em diversas plataformas, os desenvolvedores encontram uma oportunidade para aprimorar suas habilidades em programação, simultaneamente disponibilizando os jogos para avaliação pelo público.

A proposta consistia em criar um jogo de terror que posteriormente seria apresentado em uma exposição na FATEC, permitindo que os visitantes o experimentassem. Cada estudante foi encarregado de um cômodo da Mansão Assombrada, proporcionando-lhes a chance de explorar a criatividade ao desenvolverem fases usando HTML, CSS e Javascript.
